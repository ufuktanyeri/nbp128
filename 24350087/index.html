Profilim

Ben Senin Adın Soyadın, web teknolojilerine ve özellikle finans piyasalarına tutkuyla bağlı bir öğrenciyim. HTML ve CSS ile web sayfaları oluşturmayı, projeler geliştirmeyi seviyorum. Amacım, finansal okuryazarlığı artıracak ve yatırımcılara rehberlik edecek interaktif web uygulamaları geliştirmektir. Boş zamanlarımda borsa analizleri yapmayı ve ekonomik haberleri takip etmeyi severim.
